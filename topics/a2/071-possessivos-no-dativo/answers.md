# Respostas: Pronomes Possessivos

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | meinem | **mit** pede dativo; **Bruder** é masculino, então usamos **-em**. |
| 1.2 | ihrer | **bei** pede dativo; **Freundin** é feminino, então usamos **-er**. |
| 1.3 | unserem | **zu** pede dativo; **Haus** é neutro, então usamos **-em**. |
| 1.4 | deinen | **Kindern** está no plural dativo; possessivo plural usa **-en**. |
| 1.5 | seiner | **Mutter** é feminino e vem depois de **bei**, então **seiner**. |
| 1.6 | eurem | **Kurs** é masculino e vem depois de **zu**, então **eurem**. |
| 2.1 | meinem | **Vater** é masculino no dativo depois de **mit**. |
| 2.2 | ihrer | **Tante** é feminino no dativo depois de **bei**. |
| 2.3 | unseren | **Nachbarn** é plural no dativo; usa **-en**. |
| 2.4 | deiner | **Arbeit** é feminino no dativo depois de **von**. |
| 2.5 | ihrer | **Oma** é feminino no dativo depois de **bei**. |
| 2.6 | Ihrem | Forma formal de **Ihr**; **Auto** é neutro no dativo: **Ihrem**. |
| 3.1 | mit meinem Bruder | **mit** exige dativo, masculino: **meinem**. |
| 3.2 | bei deiner Schwester | **Schwester** é feminino no dativo: **deiner**. |
| 3.3 | zu unserem Haus | **Haus** é neutro no dativo: **unserem**. |
| 3.4 | mit ihren Eltern | **Eltern** é plural no dativo: **ihren**. |
| 3.5 | zu eurem Lehrer | **euer** perde um **e** na forma **eurem**. |
| 3.6 | mit seinem Kind | **Kind** é neutro no dativo: **seinem**. |
| 4.1 | Ich spreche mit meinem Bruder. | Depois de **mit**, não usamos **mein**, e sim dativo: **meinem**. |
| 4.2 | Wir sind bei unserer Lehrerin. | **Lehrerin** é feminino; no dativo fica **unserer**. |
| 4.3 | Sie fährt mit ihren Eltern. | Plural no dativo pede **ihren**. |
| 4.4 | Gehst du zu deinem Freund? | **Freund** é masculino depois de **zu**: **deinem**. |
| 4.5 | Er kommt von seiner Arbeit. | **Arbeit** é feminino depois de **von**: **seiner**. |
| 5.1 | Ich wohne bei meiner Mutter. | **bei** indica morar com/na casa de alguém e pede dativo. |
| 5.2 | Wir gehen zu unserer Schule. | **Schule** é feminino; com **zu** fica **unserer**. |
| 5.3 | Sie spricht mit ihrem Bruder. | O possuidor é **sie**; **Bruder** masculino no dativo: **ihrem**. |
| 5.4 | Kommst du mit deinen Eltern? | **Eltern** é plural no dativo: **deinen**. |
| 5.5 | Seid ihr bei eurem Lehrer? | O possuidor é **ihr**; **Lehrer** masculino no dativo: **eurem**. |
| 6.1 | Ich fahre mit meinem Bruder. | A resposta usa **mit** + dativo masculino. |
| 6.2 | Jana ist bei ihrer Freundin. | **Freundin** é feminino; depois de **bei**, usamos **ihrer**. |
| 6.3 | Wir gehen zu unserem Haus. | **Haus** é neutro; depois de **zu**, usamos **unserem**. |
| 6.4 | Ben spielt mit seinen Kindern. | Plural no dativo usa **seinen**. |
| 6.5 | Ich lerne bei meiner Lehrerin. | **Lehrerin** é feminino no dativo: **meiner**. |
