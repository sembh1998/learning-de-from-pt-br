# Respostas: Pronomes Possessivos

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Mein | `Bruder` é masculino e sujeito; no nominativo masculino, `mein` não recebe final. |
| 1.2 | Deine | `Schwester` é feminino no nominativo; o final é `-e`. |
| 1.3 | Sein | `Auto` é neutro no nominativo; `sein` fica sem final. |
| 1.4 | Ihre | `Mutter` é feminino no nominativo; `ihr` recebe `-e`. |
| 1.5 | Unsere | `Freunde` é plural; no nominativo plural, usamos `unsere`. |
| 1.6 | Euer | `Vater` é masculino e sujeito; a forma base é `euer`. |
| 2.1 | meinen | `Bruder` é masculino e objeto direto de `sehen`; acusativo masculino recebe `-en`. |
| 2.2 | deine | `Oma` é feminino; no acusativo feminino a forma é igual ao nominativo feminino. |
| 2.3 | sein | `Handy` é neutro; no acusativo neutro `sein` fica sem final. |
| 2.4 | ihre | `Schwester` é feminino; no acusativo feminino usamos `ihre`. |
| 2.5 | unsere | `Kinder` é plural; no acusativo plural do possessivo fica `unsere`. |
| 2.6 | euren | `Hund` é masculino e objeto direto de `finden`; `euer` vira `euren`. |
| 3.1 | meinen | `Vater` é masculino acusativo depois de `treffe`, por isso `meinen`. |
| 3.2 | Deine | `Tasche` é feminino e sujeito; a forma correta é `deine`. |
| 3.3 | euren | `Schlüssel` é masculino e objeto direto de `suchen`; com `euer`, fica `euren`. |
| 3.4 | Unser | `Kind` é neutro e sujeito; no nominativo neutro não há final. |
| 3.5 | ihre | `Tante` é feminino e objeto direto; no acusativo feminino é `ihre`. |
| 3.6 | sein | `Fahrrad` é neutro e objeto direto; no acusativo neutro fica `sein`. |
| 4.1 | meinen Bruder | Masculino acusativo exige final `-en` no possessivo. |
| 4.2 | deinen Vater | `Vater` é masculino; no acusativo, `dein` vira `deinen`. |
| 4.3 | seinen Hund | `Hund` é masculino; no acusativo, `sein` vira `seinen`. |
| 4.4 | ihren Lehrer | `Lehrer` é masculino; no acusativo, `ihr` vira `ihren`. |
| 4.5 | unseren Sohn | `Sohn` é masculino; no acusativo, `unser` vira `unseren`. |
| 4.6 | euren Freund | `Freund` é masculino; no acusativo, `euer` vira `euren`. |
| 5.1 | Mein Bruder wohnt hier. | `Bruder` é sujeito masculino; usamos nominativo `mein`. |
| 5.2 | Ich sehe meinen Bruder. | Depois de `sehen`, `Bruder` é objeto direto masculino; usamos `meinen`. |
| 5.3 | Ihre Schwester arbeitet heute. | `Schwester` é feminino e sujeito; `ihre` significa “dela”. |
| 5.4 | Wir besuchen unsere Freunde. | `Freunde` é plural e objeto direto; o possessivo é `unsere`. |
| 5.5 | Kennt ihr euren Lehrer? | `Lehrer` é masculino e objeto direto de `kennen`; `euer` vira `euren`. |
| 6.1 | Ich sehe meinen Vater. | `Vater` é masculino acusativo; falta o final `-en`. |
| 6.2 | Dein Bruder ist nett. | `Bruder` é masculino nominativo; não usamos `deine`. |
| 6.3 | Wir besuchen unsere Oma. | `Oma` é feminino acusativo; a forma correta é `unsere`, não `unseren`. |
| 6.4 | Eure Mutter kommt morgen. | `Mutter` é feminino nominativo; com `euer`, a forma é `eure`. |
| 6.5 | Anna ruft ihren Bruder an. | `Bruder` é masculino e objeto direto de `anrufen`; usamos `ihren`. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1. Wen besucht Clara am Samstag? | Clara besucht ihren Bruder Max. `Bruder` é masculino acusativo, por isso `ihren`. |
| 2. Onde mora o irmão de Clara? | Ele mora em um apartamento pequeno em Colônia. Na história: `Ihr Bruder wohnt in einer kleinen Wohnung in Köln.` |
| 3. Quais três coisas Clara leva? | Ela leva sua bolsa, seu celular e sua agenda nova: `ihre Tasche`, `ihr Handy`, `ihren neuen Kalender`. |
| 4. Complete em alemão: Max sucht ___ Schlüssel. | Max sucht seinen Schlüssel. `Schlüssel` é masculino acusativo. |
| 5. Por que a frase `ihren Bruder` usa `-en`? | Porque `Bruder` é masculino e objeto direto de `besucht`; acusativo masculino recebe `-en`. |
| 6. Quem traz um cachorro? | Ben traz um cachorro: `Ben bringt seinen Hund mit.` |
| 7. Complete em alemão: Sofia bringt ___ Kamera. | Sofia bringt ihre Kamera. `Kamera` é feminino acusativo, então fica `ihre`. |
| 8. Reescreva no acusativo: `euer Freund`. | `euren Freund`, porque `Freund` é masculino acusativo. |
| Prática 1 | Clara besucht ihren Bruder Max. `Bruder` é masculino acusativo. |
| Prática 2 | Ihr Bruder wohnt in Köln. Aqui `Bruder` é sujeito, então fica nominativo. |
| Prática 3 | Ben bringt seinen Hund mit. `Hund` é masculino acusativo. |
| Prática 4 | Max sagt: „Mein Schlüssel ist vielleicht in meiner Jacke.“ `Schlüssel` é sujeito, então `mein`. |
| Prática 5 | Clara bringt ihre Tasche mit. `Tasche` é feminino acusativo; não usamos `ihren`. |
| Prática 6 | Unser Schlüssel ist in der Jacke. `Schlüssel` é masculino nominativo; com `unser`, fica sem final. |
