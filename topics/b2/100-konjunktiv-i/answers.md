# Respostas: Konjunktiv I

Nível: B2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | sei | O verbo sein no Konjunktiv I, 3ª pessoa do singular, é sei. |
| 1.2 | gebe | Es gebe é a forma típica de discurso indireto para es gibt. |
| 1.3 | habe | A fonte é die Firma; no discurso indireto usa-se sie habe. |
| 1.4 | werde | Para futuro relatado, werde mantém o tom indireto formal. |
| 1.5 | könne | Können no Konjunktiv I, 3ª pessoa do singular, vira könne. |
| 1.6 | müsse | Müssen no Konjunktiv I vira müsse. |
| 2.1 | Die Bürgermeisterin sagt, die Brücke sei sicher. | Sei relata a declaração sem transformá-la em afirmação direta do redator. |
| 2.2 | Der Verein erklärt, er habe keine Schulden. | Habe é a forma esperada para relatar o que o Verein declara sobre si mesmo. |
| 2.3 | Die Polizei meldet, es gebe keine Verletzten. | Es gebe é fórmula frequente em notícias para relatar existência. |
| 2.4 | Der Trainer sagt, der Spieler könne starten. | Könne é a forma indireta de kann. |
| 2.5 | Die Forscherinnen erklären, sie bräuchten mehr Zeit. | Como sie brauchen coincide com o indicativo, bräuchten evita ambiguidade. |
| 2.6 | Der Konzern kündigt an, er werde investieren. | Werde marca futuro no discurso indireto. |
| 3.1 | Die Ministerin sagt, sie sei optimistisch. | Ich muda para sie e bin muda para sei. |
| 3.2 | Der Zeuge erklärt, er habe den Wagen gesehen. | Ich muda para er e habe marca relato indireto. |
| 3.3 | Die Behörde meldet, es gebe keine Einschränkungen. | Gibt vira gebe no Konjunktiv I. |
| 3.4 | Der Kandidat sagt, er wolle die Regeln ändern. | Will vira wolle no discurso indireto. |
| 3.5 | Die Sprecherin erklärt, sie könnten die Zahlen bestätigen. | Wir muda para sie; könnten é forma substituta clara para o plural. |
| 3.6 | Der Präsident sagt, sie müssten schneller handeln. | Wir muda para sie; müssten evita a coincidência com o indicativo. |
| 4.1 | Der Bericht schreibt, der Plan sei realistisch. | Ist vira sei para criar estilo jornalístico neutro. |
| 4.2 | Die Quelle sagt, sie habe neue Dokumente. | Hat deve virar habe no relato indireto. |
| 4.3 | Laut Feuerwehr seien alle Bewohner in Sicherheit. | Com sujeito plural, seien marca o discurso indireto. |
| 4.4 | Die Experten sagen, die Kosten könnten steigen. | Könnten é preferido quando a forma plural direta seria ambígua. |
| 4.5 | Der Verband erklärt, es gebe keinen Grund zur Sorge. | Es gebe é a forma indireta de es gibt. |
| 5.1 | discurso indireto com Konjunktiv I | Sei mostra que a informação é relatada pela Zeitung. |
| 5.2 | discurso direto | As aspas indicam fala direta. |
| 5.3 | afirmação direta no indicativo | Não há fonte relatada nem forma de Konjunktiv. |
| 5.4 | discurso indireto com Konjunktiv I | Laufe é Konjunktiv I de laufen. |
| 5.5 | afirmação direta no indicativo | Läuft é indicativo presente. |
| 6.1 | Nach Angaben der Delegation seien die Verhandlungen schwierig. | Seien relata uma avaliação atribuída à delegação. |
| 6.2 | Die Versicherung erklärte, der Schaden sei gering. | Sei substitui ist no discurso indireto. |
| 6.3 | Das Ministerium teilte mit, es gebe genug Impfstoff. | Es gebe relata a existência de vacina segundo a fonte. |
| 6.4 | Die Kommission kündigte an, sie werde die Entscheidung prüfen. | Werde expressa futuro relatado. |
| 6.5 | Die Regierung betonte, die Maßnahmen hülfen den Familien. | Hülfen é forma de Konjunktiv para helfen; também se vê würden helfen em estilo menos elevado. |
| 7.1 | Der Sprecher sagt, die Entscheidung sei endgültig. | A estrutura relata a declaração do porta-voz com sei. |
| 7.2 | Laut Polizei gebe es keine Gefahr für die Bevölkerung. | Laut Polizei combina bem com gebe para neutralidade jornalística. |
| 7.3 | Die Forscherin behauptet, sie habe neue Ergebnisse. | Habe relata a afirmação da pesquisadora. |
| 7.4 | Die Regierung kündigt an, sie werde die Steuern senken. | Werde indica ação futura anunciada. |
| 7.5 | Experten sagen, die Preise könnten sinken. | Könnten relata a avaliação de especialistas sem afirmá-la como certeza. |
