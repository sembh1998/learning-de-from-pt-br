# Respostas: Nominalização

Nível: B2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Das Lernen | Infinitivo nominalizado vira substantivo neutro com maiúscula. |
| 1.2 | Unterzeichnung | `die Unterzeichnung` é a forma formal para o ato de assinar. |
| 1.3 | Auswertung | `die Auswertung` é a análise ou avaliação de resultados. |
| 1.4 | Entscheidung | Depois de `vor der`, o substantivo feminino fica no dativo. |
| 1.5 | Durchführung | `die Durchführung` é a forma nominal comum de `durchführen`. |
| 2.1 | Die Prüfung der Dokumente erfolgt morgen. | A frase usa `die Prüfung` com complemento no genitivo e estilo formal. |
| 2.2 | Die Einführung des neuen Systems beginnt im Mai. | `die Einführung` é feminino e `des neuen Systems` expressa o complemento. |
| 2.3 | Die Anmeldung erfolgt online. | `die Anmeldung` é a nominalização natural de `sich anmelden`. |
| 3.1 | Die sorgfältige Vorbereitung der Konferenz ist wichtig. | `vorbereiten` vira `die Vorbereitung`; o adjetivo concorda com o substantivo. |
| 3.2 | Die Analyse der Daten erfolgt nach dem Interview. | A nominalização concentra a ação no substantivo `Analyse`. |
| 3.3 | Die regelmäßige Kontrolle der Anträge erfolgt durch die Behörde. | `Kontrolle` permite transformar a ação administrativa em processo formal. |
| 3.4 | Die Verbesserung der internen Prozesse erfolgt durch das Team. | `die Verbesserung` mantém o sentido de melhorar processos. |
| 4.1 | Nach der Auswertung | `nach` pede dativo; `Auswertung` é feminino: `der`. |
| 4.2 | Nach dem Abschluss | `Abschluss` é masculino; no dativo após `nach`: `dem`. |
| 4.3 | Durch das Verwenden | Depois de `durch`, o infinitivo nominalizado fica no acusativo neutro: `das`. |
| 4.4 | Bei der Einführung | `bei` pede dativo; `Einführung` é feminino: `der`. |
| 5.1 | Nach der Anmeldung erhalten Sie eine Bestätigung. | `nach` pede dativo e `Anmeldung` é feminino. |
| 5.2 | Durch regelmäßiges Üben kann man Fortschritte machen. | `Üben` é infinitivo nominalizado neutro; com adjetivo no acusativo: `regelmäßiges`. |
| 5.3 | Bei der Durchführung gab es Probleme. | `bei` pede dativo e `Durchführung` é feminino. |
| 6.1 | Die lange Diskussion der Vorschläge ist notwendig. | `Diskussion` transforma a ação de discutir em um substantivo formal. |
| 6.2 | Die frühe Planung der Veranstaltung ist sinnvoll. | `Planung` expressa o ato de planejar como processo. |
| 6.3 | Die Teilnahme der Mitarbeiter an der Schulung ist vorgesehen. | `Teilnahme an` mantém a preposição exigida pelo substantivo. |
| 6.4 | Die Zusammenarbeit mit zwei Partnern ist geplant. | `Zusammenarbeit mit` é a forma nominal natural para trabalhar em conjunto. |
| 7.1 | Das Lernen einer Sprache braucht Zeit. | Como substantivo, `Lernen` deve começar com maiúscula. |
| 7.2 | Nach der Prüfung der Unterlagen bekommen Sie eine Antwort. | `nach` pede dativo; `Prüfung` é feminino: `der`. |
| 7.3 | Die Durchführung des Projekts war teuer. | A nominalização correta de `durchführen` é `die Durchführung`. |
| 7.4 | Vor der Entscheidung sollten wir alle Fakten kennen. | `Entscheidung` é feminino; depois de `vor` temporal usa-se dativo. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1 | Porque os colegas alemães usam muitas nominalizações em relatórios, e ela quer entender esse estilo formal. |
| 2 | „Die Durchführung der Tests erfolgt im April.“ |
| 3 | O time coleta feedback de professores. |
| 4 | Ela transforma em „Die Verbesserung der Navigation hat Priorität“. |
| 5 | Porque um bom texto deve ser formal, mas também legível; nominalizações em excesso podem soar rígidas. |
| 6 | A frase verbal é mais direta e dinâmica; a nominalização soa mais objetiva, compacta e formal. |
| 7 | Durch regelmäßiges Überarbeiten wird ihr Text klarer. |
| 8 | Possível resposta: „Die Einführung neuer Funktionen erfolgt durch das Team.“ |
| Prática 1 | „Die Auswertung der Rückmeldungen ...“; `Auswertung` nominaliza `auswerten`. |
| Prática 2 | „Die vereinfachte Anmeldung ...“; `Anmeldung` vem de `sich anmelden`. |
| Prática 3 | Abschluss; a expressão é `nach dem Abschluss der Diskussion`. |
| Prática 4 | Possível resposta: „Bei der Planung achten wir auf klare Ziele.“ |
| Prática 5 | Possível resposta: „Vor der Entscheidung prüfen wir alle Daten.“ |
| Prática 6 | „Wir führen die Tests im April durch.“; é a versão verbal mais direta. |
