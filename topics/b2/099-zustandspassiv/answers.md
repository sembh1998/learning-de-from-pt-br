# Respostas: Zustandspassiv

Nível: B2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | estado | `ist geschlossen` usa `sein + Partizip II` e descreve o resultado. |
| 1.2 | processo | `wird korrigiert` usa `werden + Partizip II`, foco na ação. |
| 1.3 | estado | `sind reserviert` mostra que as reservas já existem. |
| 1.4 | processo | `wird aktualisiert` indica atualização em andamento. |
| 2.1 | sind | `Dokumente` está no plural, portanto `sind`. |
| 2.2 | ist | `Vertrag` é singular, portanto `ist`. |
| 2.3 | waren | `gestern` pede passado; `Fenster` está no plural. |
| 2.4 | ist | `Formular` é singular; `noch nicht` não muda a forma de `sein`. |
| 2.5 | sind | `Rechnungen` está no plural; `bezahlt` marca o estado. |
| 3.1 | Die Tür ist geöffnet. | Troca-se `werden` por `sein` para focar no estado. |
| 3.2 | Der Computer ist repariert. | O computador está no estado resultante de conserto. |
| 3.3 | Die Daten sind gespeichert. | `Daten` é plural, por isso `sind`. |
| 3.4 | Der Raum ist vorbereitet. | A sala está pronta; foco no resultado. |
| 3.5 | Die Anfrage ist bearbeitet. | A solicitação já está processada. |
| 4.1 | Das Paket wird verpackt. | Para processo, usa-se `werden + Partizip II`. |
| 4.2 | Die E-Mail wird verschickt. | A frase passa a indicar envio em andamento. |
| 4.3 | Die Maschine wird gereinigt. | `wird` mostra que a limpeza está acontecendo. |
| 4.4 | Die Entscheidung wird dokumentiert. | O foco muda do documento pronto para o processo de documentação. |
| 5.1 | Die Datei ist gespeichert. | O contexto fala de resultado já alcançado. |
| 5.2 | Der Vertrag wird unterschrieben. | `neste momento` exige foco no processo. |
| 5.3 | Der Raum ist vorbereitet. | A sala está pronta; isso é Zustandspassiv. |
| 5.4 | Der Aufzug wird repariert. | `está sendo consertado` pede Vorgangspassiv. |
| 6.1 | Die Tür ist geschlossen. | O contexto diz que a porta já está fechada. |
| 6.2 | Die Plätze sind reserviert. | Os lugares já estão em estado de reserva. |
| 6.3 | Der Bericht ist gedruckt. | O relatório já está impresso, não sendo impresso. |
| 6.4 | Die Aufgabe ist erledigt. | A tarefa já foi concluída; foco no resultado. |
| 7.1 | Der Vertrag ist unterschrieben. | `ist unterschrieben` expressa contrato já assinado. |
| 7.2 | Die Fenster waren geöffnet. | `waren` expressa estado no passado com plural. |
| 7.3 | Das Problem ist gelöst. | `ist gelöst` descreve o problema já resolvido. |
| 7.4 | Die Unterlagen sind vorbereitet. | `Unterlagen` é plural; usa-se `sind`. |
