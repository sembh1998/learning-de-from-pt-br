# Respostas: Discurso indireto no passado

Nível: B2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | habe | hören forma Perfekt com haben; no discurso indireto formal usa-se habe + particípio. |
| 1.2 | sei | ankommen indica deslocamento e forma Perfekt com sein; no Konjunktiv I fica sei. |
| 1.3 | hätten | No plural, haben coincide com o indicativo; hätten evita ambiguidade. |
| 1.4 | seien | fliehen indica deslocamento/mudança de lugar; no plural formal usa-se seien. |
| 1.5 | habe | sehen forma Perfekt com haben; a frase relata a fala do advogado. |
| 1.6 | sei | zurücktreten forma Perfekt com sein; a notícia relata a informação sem assumi-la diretamente. |
| 2.1 | Der Fahrer sagte, er habe sofort gebremst. | gebremst pede haben; habe marca relato formal da fala do motorista. |
| 2.2 | Die Studentin erklärte, sie sei früh nach Hause gegangen. | gehen usa sein no Perfekt; ich vira sie no relato. |
| 2.3 | Die Ermittler sagten, sie hätten Spuren gesichert. | Para plural, hätten é forma substituta clara para haben. |
| 2.4 | Der Verein teilte mit, er sei aus dem Projekt ausgestiegen. | aussteigen usa sein; Verein é singular masculino e vira er. |
| 2.5 | Die Firma gab an, sie habe den Fehler behoben. | beheben usa haben; die Firma é singular feminino e vira sie. |
| 2.6 | Die Gäste berichteten, sie seien später angekommen. | ankommen usa sein; no plural Konjunktiv I passado, seien é adequado. |
| 3.1 | Der Techniker sagte, er habe das Gerät repariert. | reparieren forma Perfekt com haben; ich muda para er. |
| 3.2 | Die Nachbarin erklärte, sie sei gegen neun zurückgekommen. | zurückkommen usa sein; ich muda para sie. |
| 3.3 | Der Politiker behauptete, er habe die E-Mail nicht gelesen. | lesen usa haben; habe mantém distância em relação à afirmação. |
| 3.4 | Die Firma teilte mit, sie habe die Produktion gestoppt. | stoppen usa haben; die Firma vira sie. |
| 3.5 | Die Besucher sagten, sie seien durch den Haupteingang gekommen. | kommen usa sein; wir vira sie. |
| 3.6 | Der Zeuge meinte, er habe lange warten müssen. | Com modal no passado, usa-se habe + infinitivo + müssen. |
| 4.1 | Die Quelle sagte, der Leiter habe die Sitzung verlassen. | verlassen usa haben; habe substitui o indicativo hat. |
| 4.2 | Die Klinik erklärte, die Patientin sei am Morgen aufgewacht. | aufwachen usa sein, portanto sei é a forma correta. |
| 4.3 | Der Bericht schreibt, mehrere Fehler seien gefunden worden. | Passiva no passado relatado: seien + Partizip II + worden. |
| 4.4 | Die Experten erklärten, sie hätten die Werte verglichen. | hätten evita confusão com o indicativo plural haben. |
| 4.5 | Der Sprecher meinte, die Verhandlungen seien gut gelaufen. | laufen usa sein no Perfekt; o plural pede seien. |
| 5.1 | relato indireto no passado | habe unterschrieben relata uma ação concluída atribuída ao mandante. |
| 5.2 | relato indireto no presente | sei sicher relata um estado presente, não uma ação passada. |
| 5.3 | afirmação direta no indicativo | dauerte afirma diretamente um fato narrado no indicativo. |
| 5.4 | relato indireto no passado | seien gerettet worden é passiva passada em discurso indireto. |
| 5.5 | afirmação direta no indicativo | hat behoben apresenta a informação diretamente no indicativo. |
| 5.6 | relato indireto no presente | gebe relata uma situação atual no Konjunktiv I. |
| 6.1 | Die Behörde teilte mit, die Kontrolle habe zwei Stunden gedauert. | dauern usa haben; a abertura exige relato formal. |
| 6.2 | Das Magazin berichtete, der Direktor sei am Freitag zurückgetreten. | zurücktreten usa sein e aparece como fato passado relatado. |
| 6.3 | Die Vertreter erklärten, sie hätten alle Dokumente abgegeben. | Plural com haben fica mais claro com hätten. |
| 6.4 | Der Händler sagte, die Ware sei gestern angekommen. | ankommen usa sein; die Ware é singular feminino. |
| 6.5 | Die Kundin erklärte, sie habe den Termin nicht verschieben können. | Modal no passado: habe + infinitivo + können. |
| 7.1 | Der Sprecher sagte, das Team habe die Zahlen geprüft. | prüfen usa haben; habe geprüft marca o passado relatado. |
| 7.2 | Laut Polizei habe der Verdächtige die Stadt verlassen. | verlassen usa haben; laut Polizei indica fonte relatada. |
| 7.3 | Die Direktorin behauptete, sie sei zu spät angekommen. | ankommen usa sein; behauptete sugere alegação. |
| 7.4 | Die Forscher sagten, sie hätten eine Lösung gefunden. | hätten é a forma substituta comum no plural. |
| 7.5 | Die Zeitung berichtete, die Sitzung sei abgesagt worden. | Passiva passada: sei + abgesagt + worden. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1. Quem relatou inicialmente a reunião no Rathaus? | Uma Lokalzeitung, isto é, um jornal local, relatou a reunião. |
| 2. O que o prefeito declarou ter feito em relação às críticas? | Ele declarou que levou as críticas a sério e já ofereceu novas conversas. |
| 3. Segundo a porta-voz, o que havia acontecido na semana anterior? | Várias reclamações haviam chegado ao órgão de trânsito. |
| 4. Qual resultado o perito teria constatado? | Ele teria constatado que os tempos de espera em dois cruzamentos ficaram claramente mais longos. |
| 5. Por que os críticos discordavam da administração? | Eles alegavam que a administração havia reagido tarde demais. |
| 6. O que a moradora disse sobre seu trajeto para o trabalho? | Ela disse que antes ia de bicicleta, mas ultimamente tinha pegado o ônibus. |
| 7. Que medida a cidade afirmou que tomaria ainda naquela semana? | A cidade afirmou que reajustaria os semáforos naquela mesma semana. |
| 8. Por que a última frase da história é importante para o tom jornalístico? | Ela mostra distância crítica: o jornal relata declarações, mas informa que elas ainda não foram verificadas independentemente. |
| Prática 1 | habe | ernst nehmen usa haben; no relato formal fica habe ernst genommen. |
| Prática 2 | seien | Beschwerden é plural e eingehen usa sein; por isso seien eingegangen. |
| Prática 3 | Die Behörde teilte mit: „Wir haben die Daten kontrolliert.“ | No discurso direto, o ponto de vista volta para wir e o indicativo haben. |
| Prática 4 | Die Anwohnerin sagte, sie sei früher mit dem Fahrrad gefahren. | fahren usa sein quando indica deslocamento; ich vira sie. |
| Prática 5 | A frase com hat reagiert afirma diretamente; a frase com habe reagiert atribui a informação aos críticos ou a uma fonte. | O Konjunktiv I cria distância jornalística. |
| Prática 6 | Laut Zeitung seien die Aussagen nicht überprüft worden. | Passiva passada relatada: seien + überprüft + worden. |
