# Gabarito: Partizip I como adjetivo

Nível: B2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | lesende | Partizip I de lesen + terminação fraca -e (die + feminino nominativo). |
| 1.2 | schreiende | Partizip I de schreien + -e (das + neutro acusativo, declinação fraca). |
| 1.3 | zitternden | Partizip I de zittern + -en (den + masculino acusativo). |
| 1.4 | steigenden | Partizip I de steigen + -en (die + plural nominativo, declinação fraca). |
| 1.5 | kreisenden | Partizip I de kreisen + -en (die + plural acusativo). |
| 2.1 | laufende | Com artigo definido der, masculino nominativo recebe -e (declinação fraca). |
| 2.2 | suchenden | Dativo feminino com einer exige a terminação -en. |
| 2.3 | schlafender | Com ein, masculino nominativo recebe -er (declinação mista). |
| 2.4 | spielenden | Plural acusativo com artigo definido die recebe -en. |
| 3.1 | Die telefonierende Frau steht am Fenster. | A relativa "die telefoniert" vira atributo: telefonierend + -e antes do substantivo. |
| 3.2 | Das weinende Kind sucht seine Mutter. | "das weint" vira weinend + -e (neutro nominativo com das). |
| 3.3 | Der ankommende Zug hat zehn Minuten Verspätung. | "der ankommt" vira ankommend + -e (masculino nominativo com der). |
| 3.4 | Wir grüßen die vorbeigehenden Nachbarn. | Plural acusativo com die: vorbeigehend + -en. |
| 4.1 | Der Hund, der bellt, gehört meinem Onkel. | O atributo bellend volta a ser oração relativa ativa no presente. |
| 4.2 | Sie fotografierte die Sonne, die untergeht. | untergehend expressa ação simultânea, por isso relativa no presente. |
| 4.3 | Die Gäste, die warten, wurden langsam ungeduldig. | wartend = "die warten"; o Partizip I é sempre ativo e simultâneo. |
| 5.1 | Der singende Mann arbeitet im Theater. | "que canta" = singend + -e com der (masculino nominativo). |
| 5.2 | Ein lächelndes Kind öffnete die Tür. | Com ein, neutro nominativo recebe -es: lächelndes. |
| 5.3 | Die schlafenden Passagiere hörten die Durchsage nicht. | Plural nominativo com die: schlafend + -en. |

## Gabarito do teste

| ID | Resposta | Explicação | Pontos |
|---|---|---|---|
| 1 | Das tanzende Paar verließ den Saal. | Paar é neutro: das + tanzende (terminação fraca -e). | 1 |
| 2 | lächelnde | Partizip I de lächeln + -e (der + masculino nominativo). | 1 |
| 3 | frierenden | Dativo neutro com dem exige a terminação -en. | 1 |
| 4 | Die fotografierenden Touristen blockieren den Eingang. | A relativa vira atributo: fotografierend + -en (plural com die). | 1 |
| 5 | protestierende | Com eine, feminino nominativo recebe -e (declinação mista). | 1 |
| 6 | kochende | Partizip I de kochen + -e (das + neutro nominativo); ação ativa: a água que ferve. | 1 |
| 7 | Die Maschine, die landet, kommt aus Lissabon. | O atributo landend equivale a uma relativa ativa no presente. | 1 |
| 8 | die joggende Frau | Feminino nominativo com die recebe -e; as outras opções têm artigo ou terminação errados. | 1 |
| 9 | Das schlafende Baby ist im Zimmer. | "que dorme" = schlafend + -e (neutro nominativo com das). | 1 |
| 10 | funkelnden | Plural acusativo com die exige -en: funkelnden. | 1 |
| 11 | em textos escritos formais | O Partizip I atributivo é típico da escrita formal; na fala usa-se a oração relativa. | 1 |
| 12 | Der schweigende Verkäufer wirkt unfreundlich. | "der schweigt" vira schweigend + -e (masculino nominativo com der). | 1 |

## Respostas das perguntas de compreensão

| Pergunta | Resposta |
|---|---|
| 1. Wo stand Lena am Freitagabend? | Sie stand auf dem überfüllten Bahnsteig des Hauptbahnhofs. |
| 2. Was machte das lächelnde Kind auf dem Bahnsteig? | Es winkte den vorbeifahrenden Zügen zu. |
| 3. Wovor warnte die laufende Durchsage? | Vor dem einfahrenden Zug auf Gleis sieben. |
| 4. Neben wem fand Lena einen Platz im Zug? | Neben einem schlafenden Studenten. |
| 5. Warum beruhigte sich das weinende Baby langsam? | Weil seine Mutter leise zu singen begann. |
| 6. Worüber schreibt Lena ihre Abschlussarbeit? | Über die wachsende Stadt Berlin. |
| 7. Diferença entre os particípios | "überfüllt" é Partizip II (estado/resultado: a plataforma já está lotada); "wartend" é Partizip I (ação ativa e simultânea: os viajantes estão esperando naquele momento). |
