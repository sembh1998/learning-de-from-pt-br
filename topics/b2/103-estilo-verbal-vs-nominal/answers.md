# Respostas: Estilo verbal vs nominal

Nível: B2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Die gründliche Analyse der Daten ist notwendig. | A ação analysieren vira o substantivo die Analyse; der Daten usa genitivo plural. |
| 1.2 | Die interne Untersuchung des Vorfalls beginnt sofort. | untersuchen vira die Untersuchung; des Vorfalls liga o objeto por genitivo. |
| 1.3 | Die Verbesserung der Kommunikation ist ein zentrales Ziel. | verbessern vira die Verbesserung; a frase fica adequada a relatório. |
| 1.4 | Die Einführung neuer Regeln ist umstritten. | einführen vira die Einführung; neuer Regeln é genitivo plural sem artigo. |
| 2.1 | Wir müssen die Kosten senken, damit das Projekt gelingt. | A nominalização die Senkung é desfeita com o verbo senken e uma oração final clara. |
| 2.2 | Alle Mitglieder müssen an der Sitzung teilnehmen. | die Teilnahme an corresponde a teilnehmen an; an der Sitzung usa dativo. |
| 2.3 | Die Gruppe diskutierte lange über die Folgen. | die Diskussion über volta para diskutieren über com a mesma preposição. |
| 2.4 | Wir setzen den Plan ab September um. | die Umsetzung corresponde ao verbo separável umsetzen. |
| 3.1 | Die Auswertung der Umfrage zeigt klare Trends. | É a opção mais formal e compacta; as outras soam mais conversacionais. |
| 3.2 | Die Teilnahme an dem Kurs ist wichtig. | Teilnahme mantém a preposição an com dativo. |
| 3.3 | Die Diskussion über die Reform war kurz. | diskutieren über gera die Diskussion über; über pede acusativo nesse uso. |
| 3.4 | Die Kritik an dem Text war ausführlich. | A forma natural é die Kritik an, não Kritisierung ou Kritikierung. |
| 3.5 | Wir bearbeiten Ihre Anfrage morgen. | Para e-mail simples, o estilo verbal é mais claro e menos burocrático. |
| 3.6 | Die Verbesserung der Prozesse ist wichtig. | A nominalização pede complemento em genitivo: der Prozesse. |
| 4.1 | Analyse | analysieren normalmente gera die Analyse. |
| 4.2 | Einführung | einführen gera die Einführung; o prefixo ein- aparece no substantivo. |
| 4.3 | Teilnahme | teilnehmen an gera die Teilnahme an. |
| 4.4 | Senkung | senken gera die Senkung. |
| 4.5 | Förderung | fördern gera die Förderung. |
| 4.6 | Kritik | kritisieren gera a forma comum die Kritik. |
| 5.1 | Die Teilnahme an der Veranstaltung ist kostenlos. | Teilnahme usa an + dativo; Veranstaltung é feminino singular: der. |
| 5.2 | Die Diskussion über den Vorschlag war intensiv. | Diskussion usa über; den Vorschlag está no acusativo. |
| 5.3 | Die Entscheidung über den Antrag fällt morgen. | A decisão sobre um pedido é normalmente Entscheidung über. |
| 5.4 | Die Verbesserung der Qualität bleibt wichtig. | O complemento da nominalização aparece no genitivo: der Qualität. |
| 5.5 | Die Kritik an dem Bericht war deutlich. | A forma natural é die Kritik an; Kritisierung soa artificial. |
| 6.1 | Die Analyse der Ergebnisse ist für das Projekt wichtig. | Resposta modelo: usa nominalização adequada e genitivo. |
| 6.2 | Die Umsetzung der Strategie erfordert klare Ziele. | Resposta modelo: Umsetzung corresponde a implementar. |
| 6.3 | Der Vergleich beider Modelle zeigt deutliche Unterschiede. | Resposta modelo: Vergleich permite condensar a ação de comparar. |
| 6.4 | Die Verbesserung der internen Kommunikation hat Priorität. | Resposta modelo: nominalização formal com complemento em genitivo. |
| 7.1 | verbal | O núcleo da ação é o verbo conjugado bewertet. |
| 7.2 | nominal | A ação aparece como substantivo: die Bewertung. |
| 7.3 | verbal | O verbo separável führen ... ein expressa a ação. |
| 7.4 | nominal | Die Einführung é a nominalização de einführen. |
| 7.5 | nominal | Durch die Modernisierung usa substantivo para expressar o meio. |
| 7.6 | verbal | modernisiert e spart são verbos conjugados. |
